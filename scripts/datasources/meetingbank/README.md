# MeetingBank Research Dataset Scripts

Scripts for working with the [MeetingBank](https://meetingbank.github.io/) research dataset.

## Data Source

- **Website**: https://meetingbank.github.io/
- **Paper**: Academic research dataset of meeting transcripts
- **Coverage**: Curated meeting transcripts for NLP research
- **Data Types**: Annotated transcripts, summaries

## Scripts

- `meetingbank_ingestion.py` - Ingest MeetingBank dataset

## Usage

```bash
python meetingbank_ingestion.py
```
